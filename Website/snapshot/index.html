POF

Perfomance Optimized Fluids

Baran Budak, Cihanser Çalışkan, İsmail Mekan
Supervisor: Dr. Gizem Kayar

System that does surface reconstruction in particle-based fluid simulation.

AIM

The main goal of the project researches whether there is a way to enhance fluid simulation. Increasing the efficiency and performance of an existing particle-based fluid simulations' surface reconstruction is a major goal. We aim to achieve these goals by implementing a variety of methods to the POF system such as using special structures to find, store particles and visualize it by using various methods like the Marching cubes.

Objectives

Many of particle-based fluid simulations today require high quality, expensive computers. In other respects in simulations we have to get closer to real-life on every aspect, In accordance with this purpose, the main objective of this project is making research and implement this research results on particle-based fluid simulations. The project focused on implement surface reconstruction of simulation in this way we research and implement various methods based on three subbranch algorithms; fast quick access to data (hash), visualization (marching cubes) and surface recognition methods. In conclusion, our projects' major objectives are making research and implement better ways of surface reconstruction on particle-based fluid simulations.

Implementation

Hasher makes access easier to particles. Surface particles are found by applying algorithm in [AIA12] in order to draw vertices of surface particles. Vertices being draw by benefiting from Marching cubes algorithm [WH87].

Implementation part consists of three parts:

Hashing

Hashing is a technique which represents particles in a 3 dimensional space converted as 1D hash table that allow very fast access on particles in the 3 dimensional space domain.

Surface Particles

This algorithm detects surface particles and the cells that particles are belongs. We can discard cells that are not used and focus on the surface particles. System works more efficiently and better performance obtained.

Marching Cubes

The algorithm is used for extracting a polygonal mesh of an isosurface from a 3D discrete scalar field. Marching cubes take the vertex data and initialize to draw by triangulation by considering 16 cases.

EVALUATION

A fluid simulation system having a difficulty to simulate too many particles. Our POF system increases the performance and simulation can handle more particles. POF system affects memory efficiency so that simulation requires less memory. The spatial hash algorithm has a huge benefit in memory efficiency. POF system makes higher frame rates possible. Surface particle recognition eliminates the cells that are not made a calculation. Surface particle cells increase the performance by discarding inactive cells. Performance and efficiency are important elements of particle-based fluid simulation.

Conclusion

The massive amounts of particles can be a computational hardship for the computer. We implement various methods to get better results by doing research. Our project focuses on catalysing computational difficulties by increasing the performance and efficiency. Project should make easier to simulate with higher quantities of particles or getting better results with the same number of particles by using combination of hash algorithm, surface particle recognition algorithm and marching cubes algorithm.
